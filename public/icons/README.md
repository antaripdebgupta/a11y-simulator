# Icons Directory

This directory should contain the extension icons:

- `icon16.png` - 16x16 pixels (toolbar icon)
- `icon48.png` - 48x48 pixels (extension management page)
- `icon128.png` - 128x128 pixels (Chrome Web Store)

## Creating Icons

You can create these icons using any image editor. Recommended:

- Use PNG format with transparency
- Follow Chrome's icon design guidelines
- Ensure icons are visually clear at all sizes

For now, you can use placeholder icons or the extension will use Chrome's default icon.
